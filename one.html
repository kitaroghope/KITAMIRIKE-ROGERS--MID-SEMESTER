WHAT IS CSS

Cascading Style Sheets: used to change or modify appearance of html elements

TYPES OF CSS

  • Inline Css

    works only on one element

  • Internal Css

    Works for only for one file

  • External Css

    Its written in an external file and can be linked to many files

Selectors

These can be used to call or identify an element to modify. Common selectors are the tag names i.e. h1, p, h2, a, div, etc

Other Selectors can be class and id which are assigned customely by the webdisigner
